FUNCIONAMIENTO DEL PROGRAMA

Este programa será utilizado para calcular el pago a realizar de una persona que adquiera un seguro.
Para calcular el pago total se concideran los siguientes aspectos: la edad del asegurado, la edad de su
conyugue (si estuviese casado), la cantidad de hijos que tenga (si los tuviese), la cantidad de propiedades 
que tiene a su nombre y el salario mensual que recibe. El programa funciona recopilando los datos antes mencionados
y aplicando un recargo por cada uno de ellos. finalmente muestra en pantalla el recargo que se aplicara y 
el total a pagar.

PARTES DEL PROBLEMA A CONSIDERAR

Las partes del problema a considerar son: 
- De que manera vamos a obtener los datos del usuario
- Las variables para la obtención de los datos
- El recargo que se aplicará por cada aspecto considerado en el monto del seguro
- La estructura condicional que utilizaremos para el proceso de lógica
- Las operaciones matematicas para el calculo del precio final
- La forma en la que el usuario pueda continuar o cerra el programa
- Mostrar los datos al usuario.

MEJORAS QUE SE PODRÍAN REALIZAR

Para mejorar el programa, principalmente se debería crear una interfáz mas amigable con el usuario y quizá también crear 
un algoritmo que en base a las propiedades a su nombre y el salario del asegurado determine si puede costear
los gastos del seguro.
